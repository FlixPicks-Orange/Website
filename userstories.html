Real World Product User Stories

Guest

  • As a guest, I wish to see the features FlixPicks has to offer.
  • As a guest, I wish to understand how FlixPicks works.
  • As a guest, I wish to know whether FlixPicks will be useful for me.
  • As a guest, I want to see what content is available.
  • As a guess, I want to see what streaming service owns recommended content.
  • As a guest, I want to see general recommendations based on popular or new media on any streaming service.
  • As a guest, I wish to use CineWheel to help friends choose media to watch.

Registered User

  • As a registered user, I need to be able to do everything that a guest can do.
  • As a registered user, I need the site to be fast and easy to use.
  • As a registered user, I need to be able to login and logout of the site using my credentials.
  • As a registered user, I need to be able to have FlixPicks randomly select a movie for me to watch.
  • As a registered user, I need to be able to see all of my subscribed-to streaming services in one place.
  • As a registered user, I need to see all content available from my subscribed-to streaming services, considering all recently added or removed content.
  • As a registered user, I need to be able to react or leave comments during the movie.
  • As a registered user, I need to be able to see the most popular parts of the movie based on watch history and other userâ€™s reactions.
  • As a registered user, I need to be able to input multiple movie options and have FlixPicks randomly select an option.
  • As a registered user, I want to be able to see a personalized library of movies based on my interests that I can manipulate.
  • As a registered user, I want to be able to search for movies or shows available through my subscriptions.
  • As a registered user, I wish to be able to reset my taste profile in the event my interests have shifted or the site is not providing quality suggestions.

Streaming Service Representative

  • As a representative of a streaming service, I want to be able to view the analytics gathered from FlixPicks.
  • As a representative of a streaming service, I want to see how often FlixPicks has recommended my content and the data surrounding user watch history.
  • As a representative of a streaming service, I want to see how many of my subscribers use FlixPicks.
  • As a representative of a streaming service, I want to see if FlixPicks has influenced someone to subscribe to my service.
  • As a representative of a streaming service, I want to see how frequently FlixPicks directs a user to my content.
  • As a representative of a streaming service, I wish to see viewing trends of my competitors so that I know when the competitors win.
  • As a representative of a streaming service, I need to be able to view analytics collected from CineMap - which points in a movie are the most watched and the most popular reactions.

Admin

  • As an admin, I need to be able to do and access everything on the site.
  • As an admin, I need to be able to see collected data on user habits; such as favorite genre, favorite streaming service, or even how often each FlixPicks Feature is used.
  • As an admin, I need to see how often and when a suggestion is ignored to help improve the CineRoll suggestion algorithm.
  • As an admin, I wish to see the same analytics that the streaming service representatives would see.
  • As an admin, I need to be able to see the analytics of FlixPicks and how many users are subscribed, signup rates, etc.
  • As an admin, I wish to collect feedback from users on satisfaction of the tools provided by FlixPicks.
  • As an admin, I need to be able to ban malicious user accounts.
  • As an admin, I need to be able to freeze/unfreeze malicious user accounts.
  • As an admin, I need to be able to see all of the comments or reactions made using CineMap.
  • As an admin, I need to reliably identify inappropriate comments among all the rest.
  • As an admin, I would like to have the power to mute users or delete comments that are inappropriate.
  • As an admin, I need to be able to ban or block users for violation of community guidelines.

Tester

  • As a tester, I need to see the bug analytic reports.
  • As a tester, I need to be able to submit bug reports.
